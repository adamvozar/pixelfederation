# CakePHP Application Skeleton

1. Install Docker Desktop
2. ```clone repository```
3. ```cd repository```
4. ```cp app/config/app_local.example.php app/config/app_local.php```
5. ```docker-compose build```
6. ```docker-compose up```
7. ```open http://localhost:8080/```
8. ```cd app```
9. run tests: ```vendor/bin/phpunit```
